Descripción del juego

Detectar Intruso es un juego diseñado para entrenar la percepción visual y la velocidad de procesamiento. El objetivo es identificar rápidamente el elemento que no pertenece al grupo.

Capacidades cognitivas trabajadas

  • Atención Selectiva
  • Control Inhibitorio
  • Velocidad Cognitiva
  • Coordinación Visomotora

¿Cómo se juega?

Aparecen elementos visuales agrupados de forma similar. El jugador debe identificar rápidamente el elemento que no pertenece al grupo.
