Atencion sostenida

Continuous Performance Task

Prueba neuropsicologica utilizada para evaluar atencion sostenida, impulsividad y control ante estimulos repetitivos.

Como funciona

  • Presenta numeros o simbolos en pantalla.
  • Hay ensayos en los que debes pulsar y otros en los que no.
  • Analiza constancia atencional e impulsividad.
  • Es muy util para estudiar concentracion sostenida.

Que puede revelar

Puede indicar dificultades para mantener la atencion, para responder a tiempo o para inhibir respuestas automaticas ante estimulos frecuentes.
