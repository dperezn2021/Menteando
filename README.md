# Título del TFG
<Desarrollo de una plataforma web interactiva para fomentar la estimulación mental en adultos>

## Autor
Nombre: Daniel Pérez Navarro  
Tutor/a: Eduardo García Pardo 
Centro: Universidad Rey Juan Carlos — Grado en Diseño y Desarrollo de Videojuegos  
Fecha: Enero/2026

## Resumen
El objetivo de este Trabajo de Fin de Grado es desarrollar una web interactiva que integre diversos minijuegos diseñados en Unity, enfocados al desarrollo y mantenimiento cognitivo en adultos.

El proyecto busca explorar cómo los videojuegos pueden contribuir al aprendizaje, la concentración, la memoria o los reflejos, a través de una experiencia accesible y atractiva visualmente. 

Se quiere conseguir que la gente adulta o más mayor pueda utilizar los videojuegos para mantener o aumentar sus capacidades cognitivas como memoria, reflejos o flexibilidad mental.
